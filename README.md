
📊 Stock Analytics Project

## Overview
This project performs end-to-end stock market analytics using:
- Python (Pandas, NumPy)
- MySQL (Apache)
- Streamlit (Visualization)
- Power BI (Dashboarding)

## Workflow
1. Raw CSV Cleaning & Extraction (50 files)
2. Metrics Calculation (8 KPIs)
3. Database Upload using Python
4. Web App Visualization using Streamlit
5. Business Dashboard using Power BI

## Tech Stack
- Python
- MySQL
- Streamlit
- Power BI

## Power BI Setup

1. Open Power BI Desktop
2. Get Data → Text/CSV
3. Load all 8 metric CSV files
4. Create relationships if needed
5. Build:
   - Top 5 Cumulative Returns
   - Volatility Comparison
   - Sector-wise Performance
6. Save as .pbix
